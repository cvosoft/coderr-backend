# Coderr
## A Fiverr clone

The [Frontend](https://github.com/cvosoft/coderr-frontend/) was realized with _VanillaJS_,  
The [Backend](https://github.com/cvosoft/coderr-backend/) was realized with _DRF_

-----

Start Frontend with  
`VS Code LiveServer`

Start Backend with  
`python3 manage.py runserver`
